# Normal Hi-Speed or Classic Hi-Speed
One question that you may have is this: Is there a difference between these two different Hi-Speed modes?

The answer is bit complex. For those who are just starting out or do not want to go in-depth into soflan journey, I would say there is no difference. The reason why is because either one of these modes are there just to make it easier for you to set your GN before turning the Floating Hi-Speed on.

But if you are interested in learning about the difference, I'll briefly explain my thoughts on both modes. Usage of each mode will be discussed more in-depth in subsequent sections.

## Classic Hi-Speed
There's actually few reasons to use Classic Hi-Speed over Normal Hi-Speed:
- Old strategies derive from plyaers using Classic Hi-Speed, which are still viable today if you prefer that style
- Allows you to easily set your GN to x1/2 or x2 value before turning on the FHS, without calculating the GN itself. This also applies if song speeds up or slows down by a constant number; it'll make calculations very easy (i.e. If a song goes from 120 BPM to 180 BPM, going from 3.00 Hi-Speed to 2.00 Hi-Speed will keep the GN the same).

## Normal Hi-Speed
Throughout my experimentation, there isn't much advantages of using Normal Hi-Speed over Classic Hi-Speed, except for maybe one reason:
- Allows you to set to your desired GN number in almost any situation, as long as you know which GN you want to use.
