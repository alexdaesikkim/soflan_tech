# Home

Welcome to Soflan Notes. This site was created with the goal of sharing IIDX Soflan related information.

## Overview

The overall goal of this site is to provide information about soflan in IIDX. This includes core concepts, techniques, as well as various examples to help you in the journey of tackling soflan charts.

**This site is not meant to be used as a "directory" for individual Soflan charts**. 

The reason being is that I want people to understand **why** these techniques exist, rather than copy and pasting a simple strategy. The reason being is that, just like how everyone uses different Green Number and White Number, each should discover a soflan strategy that suits them. By fully understanding how different techniques work and can be utilized, players should be able to explore and find something that is catered to their playstyle, rather than someone else's.

While there will be examples, these are used as an explanation on how one can approach a song rather than a definitive answer. 

If there is enough interest, I will create video examples of songs.

## Structure

The notes will be divided into different categories:

* Terminology - The terminologies that will be used throughout the site.
* Beginner - Covering the core Hi-Speed concepts.
* Normal - The fundementals techniques that you absolutely need, regardless of tacking easy or hard song.
* Hyper - Theory crafting on fundementals listed in Basic. This will include some advanced concepts such as pre-floating (theoretical ideas on how one can utilize the techniques mentioned on Basic).
* Another - Advanced theory crafting + new set of techniques that were discovered post IIDX Bistrover.
* Leggendaria - Small set of examples going over the concepts covered + few personal notes (i.e. how I tackle new or complex charts).
