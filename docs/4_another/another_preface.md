# Another (Preface)
So far, we've covered few soflan techniques, as well as some theories on how you can approach a soflan song. Up until about 2021/2022, the techniques mentioned in Normal and Hyper sections of this site were all that were known to players.

However, there were two major events that changed the soflan landscape:
- Lightning Model: With the change of the location of Start button, players had easier time hitting the start button (with some being able to hit Start and 1~7 key at the same time).
- BEMANI ProLeague: Due to the competitive nature, players were pushing their limits. Few players (i.e. CORI-, U76NER, NORI) were pushing the boundaries of soflan techniques as a result.

This resulted in two new techniques being discovered and now widely used:
- Floating (Advanced)
- x0.5 Reset

If you were wondering why I did not include (or only briefly mentioned) removing SUDDEN+ or some other techniques that are shown in other pages, the above 2 techniques made these other techniques obsolete.

This section will mostly cover these techniques. Some other techniques that are worth mentioning will be included in Misc.
